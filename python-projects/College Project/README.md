# College Project – Python Mini Projects

This folder contains my **College Project** Python work: a collection of small, self‑contained programs based on typical CS assignments (basics, graphics, games, file I/O, algorithms, grading, and recommendation systems).

Each subfolder is a standalone mini‑project with its own README and Python files.

## Project map

### Small Python basics

These are short exercises that practice one core concept each (conditionals, loops, functions, tiny games, etc.):

- `01-cafewall` – simple visual pattern.
- `02-cipher-rotation` – rotation cipher.
- `03-classes` – basic classes and objects.
- `04-complete-loop` – loop practice.
- `05-days-in-month` – days per month logic.
- `06-error-demo` – error/exception demo.
- `07-gradescope-1` – Gradescope / auto‑graded exercise.
- `08-greet-world` – greeting / intro script.
- `09-if-else-examples` – if/elif/else patterns.
- `10-input-color` – respond to color input.
- `11-input-even-odd` – even/odd checker.
- `12-is-vowel` – vowel checker.
- `13-number-square` – square of a number.
- `14-piglet-game` – tiny text game.
- `15-quadratic-equation` – quadratic solver.
- `16-quiz-4` – quiz/problem set.
- `17-return-example` – functions with return.
- `18-rotation` – rotation logic.
- `19-zune-bug` – Zune date bug style exercise.

### Larger college projects

- `02-cafewall-illusion` – visual Café Wall illusion using Python graphics and helper functions.
- `03-dna-analyzer` – simple DNA string analysis using text files for input.
- `04-file-input-output` – reading and writing files (coin flips, poem files).
- `05-gerrymandering` – vote/district analysis project (gerrymandering logic with data files).
- `06-guessing-game` – full console guessing game with lab manual.
- `07-saguro-tree` – ASCII/graphics Saguaro (cactus) tree drawing with pseudocode.
- `08-etch-a-sketch` – Etch‑A‑Sketch style drawing with keyboard input.
- `09-gradanator` – grade/score analyzer (“Grad‑anator”) with template.
- `10-recommender` – small recommendation system using ratings data and pseudocode.
